Loomkin

Elixir-native AI coding assistant. Reads your codebase, proposes edits, runs commands, commits changes — all on the BEAM.

Get Started View on GitHub

Capabilities

Everything you need to ship

27 built-in tools, multi-agent teams, and a shared nervous system that routes knowledge in real-time — backed by OTP fault tolerance.

Multi-Agent Teams

OTP-native agents coordinate through a two-layer architecture: the decision graph routes knowledge, Context Keepers preserve full-fidelity memory.

27+ Built-in Tools

File ops, code search, shell execution, Git, LSP diagnostics, decision tracking — all as Jido Actions.

Shared Nervous System

The decision graph auto-traces causality, broadcasts discoveries to relevant agents, and cascades confidence warnings downstream. Graph knows what happened — Keepers know why.

Real-Time Web UI

Phoenix LiveView streams diffs, renders decision graphs, and manages sessions — zero JavaScript required.

665+ LLM Models

Anthropic, OpenAI, Google, Groq, xAI, Bedrock and more via req_llm. Built-in cost tracking.

Ship as Binary

Burrito-wrapped single binary for macOS & Linux. No Erlang/Elixir install required.

Why the BEAM

Built different, on purpose

The BEAM VM was designed for telecom-grade concurrency and fault tolerance. Turns out that’s exactly what AI agents need.

Lightweight Processes

Thousands of concurrent tool executions on microsecond-weight BEAM processes. No thread pools, no async/await.

OTP Supervisors

When a tool crashes, the supervisor restarts it. No try/catch pyramids. The runtime handles failure.

LiveView

Real-time streaming UI rendered server-side. Diffs, graphs, and sessions update instantly over WebSocket.

Hot Code Reload

Swap tools, models, and prompts in a running system without dropping a single session or losing state.

Pattern Matching

LLM response handling via clean pattern matches instead of nested conditionals. Elixir was made for this.

Built-in Clustering

Distribute agent teams across nodes with native BEAM distribution. No service mesh required.

Quick start

Up and running in minutes

Elixir 1.18+, an API key, and six commands.

~/ — zsh
$ git clone https://github.com/bleuropa/loomkin.git $ cd loomkin $ mix setup # deps + DB $ mix phx.server # Web UI on :4200 $ mix escript.build # Build CLI binary $ ./loomkin --project . "refactor the auth module"
